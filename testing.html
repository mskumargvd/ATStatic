Software Testing & QA Program

Master the fundamentals of quality assurance, with an optional advanced module in Test Automation.

Why Learn Software Testing?

In the world of software development, quality is not just a feature—it's a necessity. QA Engineers are the guardians of quality, ensuring applications are reliable, functional, and secure. This program provides the foundational skills in Manual, API, and Database testing to start your career, with an optional pathway to the high-demand field of Test Automation.

Course Curriculum

Core Program: QA Fundamentals (3 Months)

  • Month 1: Manual Testing & Agile Methodologies

    Master the STLC, test case design, and defect reporting with Jira, all within the context of Agile and Scrum frameworks.

  • Month 2: Database & API Testing

    Learn to write SQL queries for backend data validation and master manual API testing with Postman to ensure services work as expected.

  • Month 3: Performance Testing Fundamentals

    Understand the concepts of load, stress, and scalability testing. Get hands-on with JMeter and LoadRunner to simulate user load and analyze basic performance reports.

Optional Advanced Module: Test Automation (Additional 3 Months)

  • Month 4: Intro to Test Automation with Selenium

    Learn Python for automation, set up Selenium WebDriver, and master locators to automate user actions on web applications.

  • Month 5: Building a Test Automation Framework

    Go beyond simple scripts by building a structured, reusable test framework using the Page Object Model (POM) and Pytest.

  • Month 6: Capstone: Testing in a CI/CD Pipeline

    Integrate and run your Selenium automation suite within a live Jenkins pipeline, demonstrating a complete, modern QA process from code push to test validation.
