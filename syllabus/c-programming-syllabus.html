C Programming

3-Month Foundational Program Syllabus

Month 1: Programming Fundamentals

Week 1-2: Core Concepts & Control Flow

Topics:

Set up your C compiler (GCC), learn variables, data types, operators, and master conditional logic with `if-else` and `switch-case` statements.

Lab: Write a program that prints a student's grade based on their marks.

Week 3: Loops and Functions

Topics:

Understand and implement `for`, `while`, and `do-while` loops. Learn to write modular, reusable code with functions (pass-by-value).

Lab: Create a modular calculator where each operation is a separate function.

Week 4: Pointers & Memory

Topics:

Dive into the most powerful feature of C. Understand memory addresses, the dereference operator (`*`), and pass-by-reference.

Lab: Write a function that swaps the values of two variables using pointers.

Month 2: Arrays, Strings, and Structures

Week 5-6: Arrays and Strings

Topics:

Master one-dimensional arrays, multi-dimensional arrays (matrices), and the relationship between arrays and pointers. Learn to manipulate null-terminated strings.

Lab: Write a program to find the largest number in an array and another to check for palindromes.

Week 7-8: Structures (`struct`)

Topics:

Create your own custom data types. Learn to work with arrays of structures and pointers to structures to manage complex data.

Lab: Create a `struct` to store student data and manage an array of student records.

Month 3: Advanced Topics & Capstone

Week 9: Dynamic Memory Allocation

Topics:

Go beyond static memory. Master `malloc()`, `calloc()`, and `free()` to manage memory at runtime and avoid memory leaks.

Lab: Rework the student records project to handle any number of students dynamically.

Week 10-12: File I/O & Capstone Project

Topics:

Learn to read data from and write data to text files, allowing your programs to persist information.

Capstone Project: Build a "Simple Contact Management System" that can add, view, and search contacts, and save/load the contact list to a file. This project combines structures, pointers, dynamic memory, and file handling.
