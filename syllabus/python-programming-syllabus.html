Python Programming

3-Month Foundational Program Syllabus

Month 1: Getting Started with Python

Week 1-2: Core Concepts & Control Flow

Topics:

Set up your environment, learn variables, data types, and master `if-else` statements and `for`/`while` loops.

Project: A number guessing game.

Week 3: Functions & Core Data Structures

Topics:

Learn to write clean, reusable code with functions and master Python's powerful Lists and Tuples.

Project: A command-line to-do list application.

Week 4: Advanced Data Structures

Topics:

Master Dictionaries for key-value storage and Sets for working with unique elements.

Project: A program that counts the frequency of each character in a string.

Month 2: Intermediate Concepts and Practical Skills

Week 5-6: File Handling & Error Management

Topics:

Learn to read from and write to text files. Write robust code that can handle unexpected errors gracefully using `try-except` blocks.

Project: A program that sorts names from a file and writes them to a new file.

Week 7-8: Modules and Libraries

Topics:

Understand how to use Python's built-in modules like `math` and `random`, and learn to install third-party packages using PIP.

Project: Build a simple dice rolling simulator.

Month 3: Object-Oriented Programming & Capstone

Week 9-10: Object-Oriented Programming (OOP)

Topics:

Learn the fundamentals of OOP by creating Classes and Objects. Understand core concepts like Inheritance.

Project: Create a `Student` class with attributes and methods.

Week 11-12: Capstone Project

Capstone Project:

Build a "Simple Banking Application." This command-line project will combine everything you've learned: classes, objects, functions, and data structures to create a real-world program where users can create accounts, deposit, and withdraw funds.

Career Prep: Discussion on the vast applications of Python and potential career paths.
