Data Engineering

6-Month Professional Program Syllabus

Part 1: The Data Foundation (Months 1-2)

Month 1: Advanced SQL & Python for Data Engineering

Topics:

Advanced SQL (Window Functions, CTEs). Python scripting for data processing and automation. Deep dive into Pandas for complex data transformations.

Project: Write an advanced Python script that extracts data from a database, performs complex transformations, and saves the result.

Month 2: Data Modeling & ETL Pipelines

Topics:

Data Warehousing concepts. Designing analytical schemas (Star Schema, Snowflake Schema). Building a traditional ETL pipeline from scratch using Python and PostgreSQL.

Project: Design and build a functional ETL pipeline that pulls data from multiple sources, cleans it, and loads it into a data warehouse.

Part 2: Big Data & Cloud Platforms (Months 3-4)

Month 3: Data Lakes & AWS Cloud Services

Topics:

Introduction to Big Data and the Hadoop ecosystem. The concept of a Data Lake. Using Amazon S3 as a scalable data lake. Introduction to AWS Glue for serverless ETL.

Project: Migrate the ETL pipeline from Month 2 to the cloud. Use AWS Glue to crawl data in S3 and run a transformation job.

Month 4: Distributed Processing with Apache Spark

Topics:

Introduction to Apache Spark for large-scale, in-memory data processing. Core concepts (RDDs, DataFrames, Spark SQL).

Project: Run a Spark job on Amazon EMR to process a large dataset from S3, performing complex queries and aggregations.

Part 3: Orchestration & Capstone Project (Months 5-6)

Month 5: Pipeline Orchestration & Automation

Topics:

Introduction to Apache Airflow. Writing DAGs in Python to schedule, monitor, and manage complex data pipelines. Introduction to containerization with Docker for data applications.

Project: Create an Airflow DAG that automates your AWS Glue ETL job from Month 3, scheduling it to run daily.

Month 6: Capstone Project & Interview Preparation

Capstone Project:

Build an End-to-End, Cloud-Native Data Pipeline. Ingest raw data to S3, use an EMR Spark job for transformation, load the clean data into Amazon Redshift, and orchestrate the entire pipeline with Airflow.

Career Prep: Deep dive into Data Engineering interview questions, focusing on system design for data pipelines, SQL challenges, and Python coding tests.
