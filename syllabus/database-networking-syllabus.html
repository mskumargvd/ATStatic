Database & Networking

3-Month Foundational Program Syllabus

Month 1: Computer Networking Essentials

Week 1-2: Core Networking & The TCP/IP Model

Topics:

LANs vs WANs, Client-Server Model, the 7 Layers of the OSI Model, the 4 Layers of the TCP/IP Model, Encapsulation. Hands-on with Wireshark.

Lab: Capture network traffic to identify different protocol layers.

Week 3: IP Addressing and Subnetting

Topics:

IPv4 addresses, Public vs. Private IPs, Address Classes, and the fundamentals of subnetting. Introduction to CIDR notation.

Lab: Design an IP addressing scheme for a small company network.

Week 4: Essential Network Protocols

Topics:

Deep dive into how the web works with HTTP/HTTPS, DNS (the phonebook of the internet), and DHCP. Difference between TCP and UDP.

Lab: Use command-line tools like `ping`, `nslookup`, and `traceroute` to diagnose network issues.

Month 2: Relational Databases & SQL

Week 5-6: SQL Fundamentals (DDL & DML)

Topics:

The relational model (tables, rows, columns). Keys (Primary, Foreign). Mastering `CREATE TABLE`, `INSERT`, `UPDATE`, and `DELETE` statements. Setting up MySQL.

Lab: Write SQL scripts to create a Student Information System database and insert sample data.

Week 7: Advanced SQL Queries

Topics:

Mastering the `SELECT` statement, filtering with `WHERE`, sorting with `ORDER BY`, and performing aggregations (`COUNT`, `SUM`, `AVG`) with `GROUP BY`.

Lab: Write queries to answer analytical questions about the student data.

Week 8: Joining Tables & Normalization

Topics:

Learn the power of `INNER` and `LEFT` joins to combine data from multiple tables. Understand database normalization principles (1NF, 2NF, 3NF).

Lab: Write a query that joins `students` and `courses` tables to list enrollments.

Month 3: Modern Databases & Capstone

Week 9-10: Introduction to NoSQL with MongoDB

Topics:

Explore the differences between SQL and NoSQL. Learn to model data in JSON and perform CRUD operations in MongoDB.

Lab: Insert and query student records in a MongoDB collection.

Week 11-12: Capstone Project

Capstone Project:

Design and build a simple command-line "Employee Management System." Design the database schema, write the SQL to create tables, and build a Python application that connects to the database to manage employee records.

Career Prep: Discussion on career paths like Network Engineer and Database Administrator (DBA).
