6-Month Professional Program Syllabus

Part 1: Frontend Mastery (Months 1-2)

Month 1: Modern Web Foundations

Topics:

Advanced HTML5 and CSS3 (Flexbox, Grid). Mastering core JavaScript concepts (ES6+, DOM manipulation, async/await).

Project: Build a responsive, multi-page static website for a fictional business.

Month 2: Building Interactive UIs with React

Topics:

Introduction to React, components, JSX, props, and state (`useState`). Handling user events, fetching API data (`useEffect`), and client-side routing with React Router.

Project: Build a dynamic "Movie Finder" application that uses a public API to display data.

Part 2: Backend Development with Django (Months 3-4)

Month 3: Django Fundamentals & Database Design

Topics:

The Django MVT architecture. Creating models with the Django ORM to design and interact with a PostgreSQL database. Mastering the Django Admin panel.

Project: Build a complete blog application with a backend where you can manage posts using the Django Admin.

Month 4: Building Powerful REST APIs

Topics:

Deep dive into Django REST Framework (DRF). Building API endpoints, serializing data, and handling user authentication.

Project: Add a secure REST API to the blog application. Test the API thoroughly using Postman.

Part 3: Full Stack Integration & Deployment (Months 5-6)

Month 5: Connecting Frontend & Backend

Topics:

Modify the React application to fetch data from your own Django API. Handle user login and authenticated requests. Introduction to Docker for containerization.

Project: Create a full-stack, authenticated version of your blog where users can log in via the React frontend to manage posts.

Month 6: Capstone Project & Interview Prep

Capstone Project:

Build a Mini E-commerce Platform. Create a Django backend with a REST API for users, products, and orders. Build a React frontend for browsing products and managing a shopping cart. Containerize the entire application using Docker.

Career Prep: Focus on full-stack and system design interview questions. Practice sessions on presenting the capstone project.
