Cloud & DevOps

6-Month Professional Program Syllabus

Part 1: Cloud Infrastructure Mastery (Months 1-2)

Month 1: AWS Core Services & Networking

Topics:

In-depth AWS IAM (Roles, Policies). Advanced VPC design (public/private subnets, NAT Gateways). Mastery of EC2 (instance types, AMIs, load balancing) and S3 (storage classes, lifecycle policies).

Project: Design and provision a secure, scalable, and highly available two-tier web application infrastructure on AWS manually.

Month 2: Serverless, Databases & Monitoring

Topics:

Introduction to Serverless with AWS Lambda. Managed databases with RDS. Setting up monitoring and alerting with Amazon CloudWatch and an introduction to Prometheus & Grafana.

Project: Develop a serverless function that processes image uploads to S3. Set up a CloudWatch dashboard to monitor the health of your web application.

Part 2: The DevOps & Automation Engine (Months 3-4)

Month 3: Deep Dive into Containerization

Topics:

Advanced Docker concepts (multi-stage builds, networking). Deep dive into Kubernetes architecture (Deployments, Services, ConfigMaps, Secrets, Persistent Volumes).

Project: Containerize a multi-service application (e.g., web frontend, backend API, database) and deploy it on a local Kubernetes cluster.

Month 4: The Automated Pipeline

Topics:

Mastering Jenkins for CI/CD (scripted pipelines, webhooks). In-depth Infrastructure as Code (IaC) with Terraform, managing state and modules.

Project: Create a complete Jenkins pipeline that automatically builds your Docker containers and pushes them to a registry (like Docker Hub or Amazon ECR) when code is pushed to GitHub.

Part 3: Advanced Practices & Capstone (Months 5-6)

Month 5: Configuration Management & Security

Topics:

Configuration Management with Ansible to automate server setup. Introduction to DevSecOps: integrating security scanning tools (e.g., Trivy for container scanning) into your Jenkins pipeline.

Project: Write Ansible playbooks to configure your servers. Integrate a security scanner into your CI/CD pipeline that will fail the build if vulnerabilities are found.

Month 6: Capstone Project & Interview Preparation

Capstone Project:

Build a Production-Grade, Automated Deployment Pipeline. Use Terraform to provision a Kubernetes cluster (EKS), and a Jenkins pipeline to automatically build, test, scan, and deploy a containerized application with zero downtime.

Career Prep: Extensive mock interviews for Cloud/DevOps roles, resume and portfolio optimization, and a deep dive into system design interview questions.
