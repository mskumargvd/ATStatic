Software Testing & QA

6-Month Professional Program Syllabus

Part 1: The QA Foundation (Months 1-2)

Month 1: Manual Testing & Agile Methodologies

Topics:

The SDLC vs. STLC. Test planning, test case design, and execution. Defect life cycle and professional bug reporting using Jira. Introduction to Agile and Scrum methodologies.

Project: Manually test a demo e-commerce website, identify 10-15 bugs, and write professional bug reports in a Jira project.

Month 2: Database & API Testing

Topics:

Mastering SQL for backend data validation. In-depth manual API testing using Postman to test endpoints, validate responses, and manage collections.

Project: Use Postman to test the full CRUD functionality of a sample REST API. Write SQL queries to verify that the API calls correctly updated the database.

Part 2: The Automation Engineer's Toolkit (Months 3-4)

Month 3: Introduction to Test Automation with Selenium

Topics:

Introduction to Python for test automation. Setting up Selenium WebDriver. Mastering Selenium locators (ID, Name, XPath, CSS Selectors) to find web elements. Automating common user actions.

Project: Write Selenium scripts in Python to automate the user registration and login process of a demo web application.

Month 4: Building a Test Automation Framework

Topics:

Introduction to the Page Object Model (POM) for creating reusable test code. Using a testing framework like Pytest to structure tests, run assertions, and generate reports.

Project: Convert the Selenium scripts from the previous month into a structured test automation framework using the Page Object Model and Pytest.

Part 3: Advanced Specializations & Capstone (Months 5-6)

Month 5: Performance & Intro to Security Testing

Topics:

Fundamentals of Performance Testing (Load, Stress). Creating test scripts with Apache JMeter to simulate user load. Introduction to Security Testing fundamentals, focusing on the OWASP Top 10.

Project: Create a JMeter test plan to apply a load of 100 virtual users to a demo application's login page and analyze the results.

Month 6: Capstone: Testing in a CI/CD Pipeline

Capstone Project:

Integrate and Run Automated Tests in a DevOps Pipeline. Configure a Jenkins pipeline to automatically check out code from GitHub and run your Pytest/Selenium automation suite. If tests pass, the pipeline proceeds; if they fail, the pipeline stops and sends a notification.

Career Prep: Deep dive into QA interview questions (manual, automation, SQL, and behavioral). Resume and LinkedIn optimization to highlight your new automation skills.
