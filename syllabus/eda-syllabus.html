Data Analytics with Python

3-Month Foundational Program Syllabus

Month 1: Foundations of Data Handling

Week 1-2: Data Manipulation with Pandas

Topics:

Intro to Jupyter Notebooks. Master the DataFrame, read from CSVs, and learn selecting, filtering, and sorting data.

Lab: Load and inspect a real-world dataset like the Titanic dataset.

Week 3: Data Cleaning and Preprocessing

Topics:

Tackle the most critical step of data analysis: handling missing values, correcting data types, and removing duplicates.

Lab: Take a messy, raw dataset and perform a full data cleaning process.

Week 4: Numerical Computing with NumPy

Topics:

Learn the fundamentals of the NumPy library for efficient numerical and mathematical operations on large datasets.

Lab: Solve a series of mathematical and logical problems on NumPy arrays.

Month 2: Visualizing Data and Finding Insights

Week 5-6: Data Visualization with Matplotlib & Seaborn

Topics:

Create a wide range of plots, from basic bar charts and histograms to advanced statistical visualizations like heatmaps and box plots.

Lab: Create a bar chart of survival rate by gender and a heatmap of feature correlations for the Titanic dataset.

Week 7: Data Aggregation and Grouping

Topics:

Master the powerful `.groupby()` function in Pandas to segment and analyze data by categories.

Lab: Calculate the average age and survival rate for passengers in each class of the Titanic dataset.

Week 8: Merging and Joining Datasets

Topics:

Learn to combine data from multiple sources using different types of joins (`inner`, `outer`, `left`, `right`) to create a master dataset for analysis.

Lab: Merge two separate datasets (e.g., orders and customers) into a single master table.

Month 3: Advanced Techniques & Capstone

Week 9-10: Feature Engineering & Time Series

Topics:

Create new, insightful features from existing data. Learn the basics of analyzing time-stamped data, including resampling and rolling windows.

Lab: Analyze a stock price dataset to calculate and plot its 30-day moving average.

Week 11-12: Capstone Project

Capstone Project:

Perform a full exploratory data analysis on a real-world dataset (e.g., retail sales or Zomato restaurants). You will clean the data, analyze it, create compelling visualizations, and present your findings in a comprehensive Jupyter Notebook report.

Career Prep: Discussion on the roles of a Data Analyst and Data Scientist and how to present a data project in an interview.
